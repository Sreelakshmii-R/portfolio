## 🌐 Personal Portfolio

Welcome to my personal portfolio! This website showcases my projects, technical skills, and journey as an aspiring Software Engineer. It highlights the work I've built, the technologies I use, and provides an easy way to connect with me.

## 🚀 Live Demo

🔗 **Portfolio:** https://portfolio-lime-delta-91.vercel.app/

## ✨ Features

* Responsive and modern user interface
* Smooth animations with Framer Motion
* Featured project showcase
* Skills and technology stack
* Project gallery
* Resume download
* Contact section with social links

## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/Sreelakshmii-R/portfolio.git
```

Navigate to the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
